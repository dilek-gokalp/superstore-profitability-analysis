# Superstore Profitability Analysis

## 📊 Project Overview

This project analyzes retail sales data to identify key drivers of profitability and loss.

The analysis combines Excel dashboarding and Python-based data exploration to deliver business insights.

---

## 🎯 Objectives

* Analyze profitability by category and sub-category
* Identify loss-making products
* Evaluate the impact of discounts on profit
* Explore regional performance differences
* Build a simple predictive model for loss-making orders

---

## 🛠 Tools & Technologies

* Python (Pandas, Matplotlib, Seaborn, Scikit-learn)
* Excel (Dashboard & Pivot Analysis)

---

## 📈 Key Insights

* Furniture category shows high loss rate despite strong sales
* High discounts significantly reduce profitability
* Certain sub-categories consistently generate losses
* Technology category is the most profitable
